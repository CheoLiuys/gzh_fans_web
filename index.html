安装教程

请严格按照以下步骤操作，确保插件正常安装和使用。建议将插件解压到非桌面位置，防止误删。
  1. 下载并解压插件包
    1. 点击上方"下载插件"按钮获取插件安装包（.zip文件）
    2. 将zip文件解压到本地文件夹
    3. 重要：建议解压到文档、下载或其他非桌面位置，防止误删插件文件
    4. 记住解压后的文件夹路径，后续安装需要用到
  2. 开启Chrome开发者模式
    1. 打开Chrome浏览器
    2. 在地址栏输入：chrome://extensions/
    3. 或者点击右上角三点菜单 → 更多工具 → 扩展程序
    4. 在扩展程序页面右上角打开"开发者模式"开关
  3. 加载插件
    1. 开启开发者模式后，页面会显示"加载已解压的扩展程序"按钮
    2. 点击该按钮，会弹出文件夹选择窗口
    3. 选择之前解压的插件文件夹
    4. 点击"选择文件夹"完成安装
    5. 安装成功后，插件会出现在扩展程序列表中
  4. 验证安装
    1. 登录公众号后台（mp.weixin.qq.com）
    2. 在任意页面查看是否出现粉丝查询功能
    3. 如有问题，请检查插件是否已启用
    4. 确保Chrome浏览器已更新到最新版本
注意事项：
• 请勿删除解压后的插件文件夹，否则插件将无法正常工作
• Chrome浏览器更新后可能需要重新加载插件
• 如遇到问题，请尝试重新安装或联系技术支持
